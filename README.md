## Features
:fire: TypeScript :fire:  
:fire: Three.js :fire:  
:fire: External shader files !! Not in html !! :fire:  

## How to Use
#### Install dependencies

    npm install

#### Run project

    npm run start

#### Compile code to bundle

    npm run build
